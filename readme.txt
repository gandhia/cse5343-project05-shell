In order to compile the shell program, type "make shell".
In order to run the shell program, type "./shell".

When using the shell program:
To run an executible file, please type "./FILENAME".
To copy from one file to another, type "copy FILENAME1 FILENAME2".
To delete a file, type "delete FILENAME".
To type the contents of a file onto the screen, type "type FILENAME".
To exit the shell program, type "exit".
